# GradeCheck

This app allows students to check their scores stored in a google sheets spreadsheet,
without giving direct access to it.

## Setup

- `python -m venv venv`
- `source venv/bin/activate`
- `pip install -r requirements.txt`
- `python manage.py runserver`
